# Roadmap

## High Priority

- Pinch-to-zoom for board viewing on mobile
- AI difficulty levels (easy / medium / hard simulation counts)

## Medium Priority

- Game timer / time controls
- Game state sharing / export (shareable links)
- Tutorial / help system for new players
- Sound effects and move animations
- Import games from notation strings

## Low Priority

- Online multiplayer (WebSocket)
- Game replay system
- Statistics dashboard
- ELO rating system
- Board themes and customization
- Tournament mode
- Accessibility improvements
- AI analysis mode (show evaluation during play)

## Technical

- Web Worker for MCTS to prevent UI blocking
- TypeScript migration
- Bundle size optimization (lazy load MCTS engine)
- Opening book for AI
